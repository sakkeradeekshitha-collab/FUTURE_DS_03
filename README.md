# Marketing Funnel & Conversion Performance Analysis

## Objective

Analyze a marketing funnel to identify conversion rates, drop-off points, and opportunities for optimization.

## Tools

* Python
* Pandas
* Matplotlib
* VS Code

## Results

* Total Contacts: 45,211
* Leads: 32,191
* Customers: 5,289
* Traffic → Lead Conversion: 71.20%
* Lead → Customer Conversion: 16.43%

## Files

* analysis.py
* funnel_dashboard.png
* report.pdf

## Key Insight

The largest drop-off occurs between the lead and customer stages, indicating the need for improved lead nurturing strategies.
